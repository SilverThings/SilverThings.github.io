Silverspoon.io

Simplistic IoT Platform

Welcome to Silverspoon.io

Silverspoon.io is an open source project and a member of the SilverWare family.

Aim of this project is to provide users with an easy-to-use development platform for creating complex IoT solutions.
The basic building block of Silverspoon.io is Apache Camel, a light-weight integration framework based on well known Enterprise Integration Patterns. With Apache Camel, we are able to easily integrate capabilities of embedded devices with external IT systems, or support rule and event based decision making.

Features

Bulldog Library

Java library providing developers with GPIO and low-level IO capabilities of embedded linux platforms like I2C, SPI, or PWM. Currently supports Raspberry Pi, BeagleBoneBlack and CubieBoard.

Learn More

Apache Camel

Silverspoon.io consists of several pluggable component for Apache Camel. This allows us to easily integrate various parts and functionalities of IoT Gateways with other systems.

Learn More

Web Based IDE

We expect the IDE to be based on Hawt.io and to support developing custom Camel application in a web browser, as well as generating recommended wiring diagrams.
